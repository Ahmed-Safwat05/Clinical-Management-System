## 🎓 Key Concepts

### Cancelled Appointments
- Soft-deleted (not removed)
- Hidden from active queue
- Trigger reordering
- Preserved in database

### Queue Reordering
- Automatic after cancellation
- Same doctor, date, type only
- No gaps in numbering
- Efficient implementation

### Walk-In Detection
- Automatic (date == today)
- Persistent in database
- Independent numbering
- Visual distinction (W prefix)

### Separate Queues
- Scheduled: 1, 2, 3, 4...
- Walk-In: W1, W2, W3, W4...
- Per doctor
- Independent management
